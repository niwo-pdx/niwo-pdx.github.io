# Nico's Portfolio Site

Deployed site: https://niwo-pdx.github.io/

All code for the site is located in this repo. This site is still under development; updates will be documented here in the future.

## How to Run This Code (Locally)

- Download this repo and open with preferred IDE (VS Code is what I used)
- If using VS Code, download Live Server extension
- Right click on HTML and select "Go Live" (can also select this from the bottom right of VS Code)
